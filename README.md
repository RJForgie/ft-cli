# FT-CLI

FT-CLI is a node base console application that fetches current headlines from the Financial Times.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install FT-CLI.

```
npm install ft-cli
```

## Usage

```
npm run ft-cli
```
